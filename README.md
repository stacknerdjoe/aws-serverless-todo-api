# AWS Serverless Todo API

A serverless REST API for managing todos, built on AWS Lambda, API Gateway, DynamoDB, and S3, with authentication through AWS Cognito.

This project started from a course scaffold and has since been rebuilt: migrated to AWS SDK v3, moved authentication from Auth0 to Cognito, upgraded to Node.js 22, added full error handling, and added a Jest test suite with 100% coverage on the core logic.
